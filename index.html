22 ชนิดสมุนไพรไทยเสี่ยงสูญพันธุ์ !! มีการประกาศคุ้มครองเเละควบคุม 7 ชนิด




" กระทรวงสาธารณสุข (สธ.) ประกาศชื่อสมุนไพรไทย 22 ชนิดที่เสี่ยงต่อการสูญพันธุ์ในปัจจุบัน เนื่องจากสมุนไพรบางชนิดมีการนำมาใช้งานบ่อย หรือบางชนิดจำเป็นต้องใช้ทั้งต้นและราก ทำให้เติบโตช้าไม่ทันใช้งาน ซึ่งอาจนำไปถึงการสูญพันธ์ได้ โดยมีการคัดเลือกมาจากคณะผู้เชี่ยวชาญที่เข้ามาจัดทำข้อมูลของสมุนไพร 3 ประเภทของแต่ละจังหวัด ได้แก่ สมุนไพรเพื่อการศึกษา สมุนไพรพืชเศรษฐกิจ และสมุนไพรเสี่ยงอาจสูญพันธ์ และคณะกรรมการคุ้มครองและส่งเสริมภูมิปัญญาการแพทย์แผนไทยยังได้ทำการหารือคัดเลือก ประกาศคุ้มครองและควบคุมสมุนไพร 7 ชนิดอีกด้วย ซึ่งได้แก่ ถั่วดินโคก เทพทาโร มะตูมนิ่ม มะหาด เร่ว หัวร้อยรู จากสมุนไพรไทยที่เสี่ยงสูญพันธ์ 22 ชนิดและมีการเสนอชื่อกระวานเพิ่มด้วย โดยผู้ที่มีในครอบครองไม่ถือว่ามีความผิด แต่จะต้องแจ้งจำนวนที่มีในครอบครองให้สาธารณะสุขทราบ เพื่อทำแผนการใช้งานและการปลูกทดแทนต่อไป "



สรรพคุณของสมุนไพรทั้ง 22 ชนิด + "กระวาน" ที่มีการเสนอชื่อให้ทำการคุ้มครองและควบคุม



ถั่วดินโคก

สรรพคุณ : นำมาฝนกับน้ำดื่มเป็นยาช่วยลดไข้ในเด็ก ใช้ต้มกับน้ำดื่มเป็นยาบำรุงน้ำนมของสตรี (กินมากไม่ดี) และใช้ลำต้น ราก เหง้า และใบนำมาฝนทาเป็นยาแก้ฝี

เทพทาโร

สรรพคุณ : แก้ไข้หวัด แก้ไอ อาการไอเรื้อรัง ตัวร้อน นิยมนำมาทำยาหอมเพื่อแก้อาการจุกเสียด แน่น เฟ้อ แก้อาการปวดท้อง บำรุงธาตุ ขับลมในลำไส้ และกระเพาะอาหาร

มะตูมนิ่ม

สรรพคุณ : คนโบราณมักนำมาเป็นยาอายุวัฒนะเพื่อบำรุงกำลัง เสริมสรรมถภาพทางเพศ ชลอความแก่ทำให้ผมหงอกช้า บำรุงน้ำเหลืองและรักษาน้ำดี แก้อาการปวดหัว

มะหาด

สรรพคุณ : แก่นมะหาดมีรสร้อนเป็นยาแก้กระษัย ละลายเลือด กระจายโลหิต เปลือกต้นสดนำมาต้มกับน้ำกินเป็นยาแก้ไข้ หรือใช้รากสดหรือแห้ง นำมาต้มกับน้ำกินเป็นยาแก้ไข้

เร่ว

สรรพคุณ : ช่วยบำรุงธาตุในร่างกาย ด้วยการใช้เมล็ดเร่วผสมกับหัวแห้วหมู ขิงแห้ง และชะเอมเทศ นำมาปรุงเป็นยารับประทาน ผลมีรสเผ็ดปร่าใช้เป็นยาแก้ไข้และช่วยแก้ธาตุพิการ

หัวร้อยรู

สรรพคุณ : ช่วยบำรุงหัวใจ ช่วยขับชีพจร ใช้เป็นยารักษาเบาหวาน ใช้ตำกินเป็นยาขับพยาธิ ใช้เป็นยาแก้พิษประดงหรืออาการของโรคผิวหนังที่มีผื่นคันเป็นเม็ดขึ้นคล้ายผด

กระวาน

สรรพคุณ : แก้ท้องอืด ท้องเฟ้อ จุกเสียดแน่นท้อง ลดไข้ แก้เสมหะ ช่วยต้านเชื้อไข้มาลาเรีย มีฤทธิ์ยับยั้งอาการภูมิคุ้มกันบกพร่อง ต้านเซลล์มะเร็งได้

กระทุ่มนา

สรรพคุณ : ใบมีรสขมเฝื่อนเมา มีสรรพคุณช่วยลดความดันโลหิตและออกฤทธิ์กดต่อประสาทและกล้ามเนื้อ

ขันทองพยาบาท

สรรพคุณ : ช่วยแก้ลมและโลหิตเป็นพิษ แก้โรคตับ ช่วยแก้ปอดพิการ ใช้เป็นยาบำรุงเหงือก ใช้รักษาเหงือกอักเสบ ทำให้ฟันทน เหงือกแข็งแรง

จุกโรหินี

สรรพคุณ : ผลนำมาต้มกับน้ำดื่ม จะช่วยลดระดับน้ำตาลในเลือดได้ ช่วยแก้อาการอ่อนเพลีย ช่วยบำรุงกำลัง

ชะเอมไทย

สรรพคุณ : เปลือกต้นใช้ต้มกับน้ำดื่มช่วยแก้อาการไอ หากอาการไม่ดีขึ้นให้รับประทานติดต่อกัน 2-4 วัน ช่วยขับเสมหะ แก้น้ำลายเหนียว

ชิงชี่

สรรพคุณ : ใบนำมาต้มดื่มเป็นยาแก้ไข้ รากใช้เป็นยาขับปัสสาวะ ช่วยทำให้มดลูกเข้าอู่

ตับเต่า

สรรพคุณ : เปลือกใช้เป็นยารักษาโรครำมะนาด ใช้ปรุงเป็นยาแก้ท้องร่วง รากใช้ปรุงเป็นยารักษาแผลเรื้อรัง ฝนกินเป็นยาแก้ไข้ ลดไข้ ดับพิษร้อน แก้ร้อนใน แก้พิษไข้ แก้พิษทั้งปวง

นางแย้มป่า

สรรพคุณ : เปลือกใช้เป็นยารักษาโรครำมะนาด น้ำต้มจากแก่นและรากมีสรรพคุณเป็นยาบำรุงปอด

ปลาไหลเผือก

สรรพคุณ : รากใช้เป็นยาบำรุงกำลัง บำรุงร่างกาย ใช้เป็นยาบำรุงโลหิต ช่วยรักษาความดันโลหิตสูง รากมีรสขม เบื่อเมาเล็กน้อย ใช้เป็นยาถ่ายพิษต่าง ๆ ทุกชนิด

พังคี

สรรพคุณ : รากพังคีช่วยแก้อาการจุกเสียด แก้ท้องอืด ท้องเฟ้อ ปวดท้อง รากใช้ตำประคบแก้อาการปวด

มะคังแดง

สรรพคุณ : ใช้เป็นยาแก้ไข้ เปลือกต้นใช้เข้ายากับมุ่ยขาว ทำเป็นยาลูกกลอน แก้ริดสีดวงทวาร เนื้อไม้มีรสเย็นเฝื่อน ใช้ต้มกับน้ำดื่มเป็นยาแก้เลือดลมเดินไม่สะดวก แก้พิษโลหิตและน้ำเหลืองเปลือกต้นใช้ตำพอกแผลสดเพื่อห้ามเลือด

สะค้าน

สรรพคุณ : เครือใช้ประกอบอาหารช่วยเพิ่มรสเผ็ด ลำต้นใช้ใส่แกง ช่วยให้มีกลิ่นหอม รากมีสรรพคุณเป็นยาแก้ไข้ ยาแก้หืด

สารภีป่า

สรรพคุณ : ใช้เป็นยาถ่ายพยาธิ ดอกมีสรรพคุณเป็นยาบำรุงหัวใจ มีสารช่วยขยายหลอดลม และช่วยขับลม เปลือกเป็นยาแก้บิด

อบเชยไทย

สรรพคุณ : เป็นยาร้อนออกฤทธิ์ต่อไต ม้าม และกระเพาะปัสสาวะ ใช้เป็นยาบำรุงร่างกาย ทำให้ร่างกายอบอุ่น ช่วยกระจายความเย็นในร่างกาย ทำให้เลือดหมุนเวียนดี

เฉียงพร้านางแอ

สรรพคุณ : ลำต้นใช้ต้มน้ำดื่มช่วยบำรุงร่างกาย ทำให้เจริญอาหารหรือใช้เป็นยาเจริญอาหารสำหรับสตรีหลังคลอดบุตร

เถาเอ็นอ่อน

สรรพคุณ : เป็นยาเย็น มีพิษ ออกฤทธิ์ต่อหัวใจและตับ ใช้เป็นยาฟอกเลือด ขับลมในลำไส้และในกระเพาะอาหาร ทำให้ผายลมและเรอ ช่วยแก้อาการจุกเสียดแน่นท้อง

เปราะหอม

สรรพคุณ : ใช้เป็นยาแก้ปวดศีรษะ คลายเครียด ใช้เป็นยาบำรุงหัวใจ บำรุงประสาท ด้วยการใช้หัวผสมลงในยาหอม หัวเปราะหอมนำมาต้มหรือชงกิน จะช่วยในการนอนหลับได้เป็นอย่างดี และยังช่วยลดความเครียดได้อีกด้วย
